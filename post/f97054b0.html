1.В BMP еӣҫеғҸд»Ӣз»Қ

жҲ‘д»¬еёёз”Ёзҡ„еӣҫзүҮж јејҸжңүеҫҲеӨҡпјҢдёҖиҲ¬жңҖеёёз”Ёзҡ„жңүдёүз§Қпјҡ JPEGпјҲжҲ– JPGпјүгҖҒ PNGгҖҒ BMP е’Ң GIFгҖӮе…¶дёӯ JPEGпјҲжҲ– JPGпјүгҖҒ PNG д»ҘеҸҠ BMP йғҪжҳҜйқҷжҖҒеӣҫзүҮпјҢиҖҢ GIF еҲҷеҸҜд»Ҙе®һзҺ°еҠЁжҖҒеӣҫзүҮгҖӮ

BMPпјҲе…Ёз§° BitmapпјүжҳҜ Window ж“ҚдҪңзі»з»ҹдёӯзҡ„ж ҮеҮҶеӣҫеғҸж–Үд»¶ж јејҸпјҢж–Үд»¶еҗҺзјҖеҗҚдёәвҖң.bmpвҖқпјҢдҪҝз”Ёйқһеёёе№ҝгҖӮе®ғйҮҮз”ЁдҪҚжҳ е°„еӯҳеӮЁж јејҸпјҢйҷӨдәҶеӣҫеғҸж·ұеәҰеҸҜйҖүд»ҘеӨ–пјҢеӣҫеғҸж•°жҚ®жІЎжңүиҝӣиЎҢд»»дҪ•еҺӢзј©пјҢеӣ жӯӨпјҢ BMP еӣҫеғҸж–Үд»¶жүҖеҚ з”Ёзҡ„з©әй—ҙеҫҲеӨ§пјҢдҪҶжҳҜжІЎжңүеӨұзңҹгҖҒ е№¶дё”и§Јжһҗ BMP еӣҫеғҸз®ҖеҚ•гҖӮ

BMP ж–Үд»¶зҡ„еӣҫеғҸж·ұеәҰеҸҜйҖү lbitгҖҒ 4bitгҖҒ 8bitгҖҒ 16bitгҖҒ 24bit д»ҘеҸҠ 32bitпјҢ е…ёеһӢзҡ„ BMP еӣҫеғҸж–Үд»¶з”ұеӣӣйғЁеҲҶз»„жҲҗпјҡ

в‘ гҖҒ BMP ж–Үд»¶еӨҙпјҲBMP file headerпјүпјҢе®ғеҢ…еҗ« BMP ж–Үд»¶зҡ„ж јејҸгҖҒеӨ§е°ҸгҖҒ дҪҚеӣҫж•°жҚ®зҡ„еҒҸз§»йҮҸзӯүдҝЎжҒҜпјӣ

в‘ЎгҖҒдҪҚеӣҫдҝЎжҒҜеӨҙпјҲbitmap informationпјү пјҢе®ғеҢ…еҗ«дҪҚеӣҫдҝЎжҒҜеӨҙеӨ§е°ҸгҖҒ еӣҫеғҸзҡ„е°әеҜёгҖҒ еӣҫеғҸеӨ§е°ҸгҖҒ дҪҚе№ійқўж•°гҖҒеҺӢзј©ж–№ејҸд»ҘеҸҠйўңиүІзҙўеј•зӯүдҝЎжҒҜпјӣ

в‘ўгҖҒи°ғиүІжқҝпјҲcolor paletteпјүпјҢиҝҷйғЁеҲҶжҳҜеҸҜйҖүзҡ„пјҢеҰӮжһңдҪҝз”Ёзҙўеј•жқҘиЎЁзӨәеӣҫеғҸпјҢ и°ғиүІжқҝе°ұжҳҜзҙўеј•дёҺе…¶еҜ№еә”йўңиүІзҡ„жҳ е°„иЎЁпјӣ

в‘ЈгҖҒдҪҚеӣҫж•°жҚ®пјҲbitmap dataпјүпјҢд№ҹе°ұжҳҜеӣҫеғҸж•°жҚ®гҖӮ

BMP ж–Үд»¶еӨҙгҖҒдҪҚеӣҫдҝЎжҒҜеӨҙгҖҒи°ғиүІжқҝе’ҢдҪҚеӣҫж•°жҚ®пјҢ жҖ»з»“еҰӮдёӢиЎЁжүҖзӨәпјҡ

ж•°жҚ®ж®өеҗҚз§° еӨ§е°ҸпјҲByteпјү иҜҙжҳҺ
bmp ж–Үд»¶еӨҙ (bmp file header) 14 еҢ…еҗ« BMP ж–Үд»¶зҡ„ж јејҸгҖҒеӨ§е°ҸгҖҒеҲ°дҪҚеӣҫж•°жҚ®зҡ„ еҒҸз§»йҮҸзӯүдҝЎжҒҜ
дҪҚеӣҫдҝЎжҒҜеӨҙ (bitmap informationпјү йҖҡеёёдёә 40 жҲ– 56 еӯ—иҠӮ еҢ…еҗ«дҪҚеӣҫдҝЎжҒҜеӨҙеӨ§е°ҸгҖҒ еӣҫеғҸзҡ„е°әеҜёгҖҒеӣҫеғҸеӨ§ е°ҸгҖҒдҪҚе№ійқўж•°гҖҒ еҺӢзј©ж–№ејҸд»ҘеҸҠйўңиүІзҙўеј•зӯүдҝЎ жҒҜпјӣ
и°ғиүІжқҝ (color palette) з”ұйўңиүІзҙўеј•ж•°еҶіе®ҡ еҸҜйҖүпјҢеҰӮжһңдҪҝз”Ёзҙўеј•жқҘиЎЁзӨәеӣҫеғҸзҡ„йўңиүІпјҢ еҲҷи°ғ иүІжқҝе°ұжҳҜзҙўеј•дёҺе…¶еҜ№еә”йўңиүІзҡ„жҳ е°„иЎЁпјӣ
дҪҚеӣҫж•°жҚ® (bitmap dataпјү з”ұеӣҫеғҸе°әеҜёеҶіе®ҡ еӣҫеғҸж•°жҚ®

дёҖиҲ¬еёёи§Ғзҡ„еӣҫеғҸйғҪжҳҜд»Ҙ 16 дҪҚпјҲRгҖҒ GгҖҒ B дёүз§ҚйўңиүІеҲҶеҲ«дҪҝз”Ё 5bitгҖҒ 6bitгҖҒ 5bit жқҘиЎЁзӨәпјүгҖҒ 24 дҪҚпјҲRгҖҒ GгҖҒB дёүз§ҚйўңиүІйғҪдҪҝз”Ё 8bit жқҘиЎЁзӨәпјү иүІеӣҫеғҸдёәдё»пјҢжҲ‘д»¬з§°иҝҷж ·зҡ„еӣҫеғҸдёәзңҹеҪ©иүІеӣҫеғҸпјҢ зңҹеҪ©иүІеӣҫеғҸжҳҜдёҚйңҖиҰҒи°ғиүІжқҝзҡ„пјҢеҚідҪҚеӣҫдҝЎжҒҜеӨҙеҗҺйқўзҙ§и·ҹзҡ„е°ұжҳҜдҪҚеӣҫж•°жҚ®дәҶгҖӮ

еҜ№жҹҗдәӣ BMP дҪҚеӣҫж–Үд»¶иҜҙе№¶йқһеҰӮжӯӨпјҢ еҰӮ 16 иүІдҪҚеӣҫгҖҒ 256 иүІдҪҚеӣҫпјҢе®ғд»¬йңҖиҰҒдҪҝз”ЁеҲ°и°ғиүІжқҝпјҢе…·дҪ“и°ғиүІжқҝеҰӮдҪ•дҪҝз”ЁпјҢжҲ‘д»¬дёҚе…іеҝғпјҢиҝҷйҮҢжҲ‘д»¬е°Ҷдјҡд»Ҙ 16 дҪҚиүІпјҲRGB565пјү BMP еӣҫеғҸдёәдҫӢиҝӣиЎҢеӯҰд№ гҖӮ

2. BMPеӣҫеғҸж јејҸ

д»ҘдёҖеј  16 дҪҚ BMP еӣҫеғҸдёәдҫӢпјҢеҰӮдёӢеӣҫжүҖзӨәпјҡ

йҰ–е…ҲеңЁ Windows дёӢжҹҘзңӢиҜҘеӣҫзүҮзҡ„еұһжҖ§пјҢеҰӮдёӢжүҖзӨәпјҡ

еҸҜд»ҘзңӢеҲ°иҜҘеӣҫзүҮзҡ„еҲҶиҫЁзҺҮдёә 800*480пјҢдҪҚж·ұеәҰдёә 16bitпјҢжҜҸдёӘеғҸзҙ зӮ№дҪҝз”Ё 16 дҪҚиЎЁзӨәпјҢд№ҹе°ұжҳҜ RGB565гҖӮжҺҘдёӢжқҘдҪҝз”ЁеҚҒе…ӯиҝӣеҲ¶жҹҘзңӢе·Ҙе…·е°Ҷ image.bmp ж–Үд»¶жү“ејҖпјҢж–Үд»¶еӨҙйғЁеҲҶзҡ„еҶ…е®№еҰӮдёӢжүҖзӨәпјҡ

1. bmpж–Үд»¶еӨҙ

Windows дёӢдёә bmp ж–Үд»¶еӨҙе®ҡд№үдәҶеҰӮдёӢз»“жһ„дҪ“пјҡ

з»“жһ„дҪ“дёӯжҜҸдёҖдёӘжҲҗе‘ҳиҜҙжҳҺеҰӮдёӢпјҡ

еҸҳйҮҸеҗҚ ең°еқҖеҒҸз§» еӨ§е°Ҹ дҪңз”Ё
bfType 00H 2 bytes иҜҙжҳҺ bmp ж–Үд»¶зҡ„зұ»еһӢпјҢеҸҜеҸ–еҖјдёәпјҡ в‘ BM вҖ“ Windows в‘ЎBA вҖ“ OS/2 Bitmap Array в‘ўCI вҖ“ OS/2 Color Iconв‘ЈCP вҖ“ OS/2 Color Pointer в‘ӨIC вҖ“ OS/2 Icon в‘ҘPT вҖ“ OS/2 Pointer
bfSize 02H 4 bytes иҜҙжҳҺиҜҘж–Үд»¶зҡ„еӨ§е°ҸпјҢд»Ҙеӯ—иҠӮдёәеҚ•дҪҚгҖӮ
bfReserved1 06H 2 bytes дҝқз•ҷеӯ—ж®өпјҢеҝ…йЎ»и®ҫзҪ®дёә 0гҖӮ
bfReserved2 08H 2 bytes дҝқз•ҷеӯ—ж®өпјҢеҝ…йЎ»и®ҫзҪ®дёә 0гҖӮ
bfOffBits 0AH 4 bytes иҜҙжҳҺд»Һж–Үд»¶иө·е§ӢдҪҚзҪ®еҲ°еӣҫеғҸж•°жҚ®д№Ӣй—ҙзҡ„еӯ—иҠӮеҒҸз§»йҮҸгҖӮ иҝҷдёӘеҸӮж•°йқһеёёжңүз”ЁпјҢеӣ дёәдҪҚеӣҫдҝЎжҒҜеӨҙе’Ңи°ғиүІжқҝзҡ„й•ҝеәҰ дјҡж №жҚ®дёҚеҗҢзҡ„жғ…еҶөиҖҢеҸҳеҢ–пјҢжүҖд»ҘжҲ‘д»¬еҸҜд»Ҙз”ЁиҝҷдёӘеҒҸз§» йҮҸиҝ…йҖҹд»Һж–Үд»¶дёӯжүҫеҲ°еӣҫеғҸж•°жҚ®зҡ„еҒҸз§»ең°еқҖгҖӮ

bmp ж–Үд»¶еӨҙзҡ„еӨ§е°Ҹеӣәе®ҡдёә 14 дёӘеӯ—иҠӮгҖӮ д»ҺдёҠйқўзҡ„жҸҸиҝ°дҝЎжҒҜпјҢеҶҚжқҘеҜ№з…§ж–Үд»¶ж•°жҚ®пјҡ

00~01Hпјҡ 0x42гҖҒ 0x4D еҜ№еә”зҡ„ ASCII еӯ—з¬ҰеҲҶеҲ«дёәдёә BгҖҒ MпјҢиЎЁзӨәиҝҷжҳҜ Windows жүҖж”ҜжҢҒзҡ„дҪҚеӣҫж јејҸпјҢиҜҘеӯ—ж®өеҝ…йЎ»жҳҜвҖңBMвҖқжүҚжҳҜ Windows дҪҚеӣҫж–Үд»¶гҖӮ

02~05Hпјҡ еҜ№еә”дәҺж–Үд»¶еӨ§е°ҸпјҢ 0x000BB848=768072 еӯ—иҠӮпјҢдёҺ image.bmp ж–Үд»¶еӨ§е°ҸжҳҜзӣёз¬Ұзҡ„гҖӮ

06~09Hпјҡ дҝқз•ҷеӯ—ж®өгҖӮ

0A~0Dпјҡ 0x00000046=70пјҢеҚід»Һж–Үд»¶еӨҙйғЁејҖе§ӢеҲ°дҪҚеӣҫж•°жҚ®йңҖиҰҒеҒҸз§» 70 дёӘеӯ—иҠӮгҖӮ

еҗҢж ·пјҢ Windows дёӢдёәдҪҚеӣҫдҝЎжҒҜеӨҙе®ҡд№үдәҶеҰӮдёӢз»“жһ„дҪ“пјҡ

з»“жһ„дҪ“дёӯжҜҸдёҖдёӘжҲҗе‘ҳиҜҙжҳҺеҰӮдёӢ :

еҸҳйҮҸеҗҚ ең°еқҖеҒҸз§» еӨ§е°Ҹ дҪңз”Ё
biSize 0EH 4 bytes дҪҚеӣҫдҝЎжҒҜеӨҙеӨ§е°ҸгҖӮ
biWidth 12H 4 bytes еӣҫеғҸзҡ„е®ҪеәҰпјҢд»ҘеғҸзҙ дёәеҚ•дҪҚгҖӮ
biHeight 16H 4 bytes еӣҫеғҸзҡ„й«ҳеәҰпјҢд»ҘеғҸзҙ дёәеҚ•дҪҚгҖӮ жіЁж„ҸпјҢиҝҷдёӘеҖјйҷӨдәҶз”ЁдәҺжҸҸиҝ°еӣҫеғҸзҡ„й«ҳеәҰд№ӢеӨ–пјҢ е®ғиҝҳжңүеҸҰеӨ–дёҖдёӘз”ЁйҖ”пјҢз”ЁдәҺжҢҮжҳҺиҜҘеӣҫеғҸжҳҜеҖ’еҗ‘ зҡ„дҪҚеӣҫгҖҒиҝҳжҳҜжӯЈеҗ‘зҡ„дҪҚеӣҫгҖӮ еҰӮжһңиҜҘеҖјжҳҜдёҖдёӘжӯЈж•°пјҢиҜҙжҳҺжҳҜеҖ’еҗ‘зҡ„дҪҚеӣҫпјӣеҰӮжһңиҜҘеҖјжҳҜдёҖдёӘиҙҹж•°пјҢеҲҷиҜҙжҳҺжҳҜжӯЈеҗ‘зҡ„дҪҚеӣҫгҖӮ дёҖиҲ¬жғ…еҶөдёӢпјҢ BMP еӣҫеғҸйғҪжҳҜеҖ’еҗ‘зҡ„дҪҚеӣҫпјҢд№ҹе°ұжҳҜиҜҘеҖјжҳҜдёҖдёӘжӯЈж•°гҖӮ
biPlanes 1AH 2 bytes иүІеҪ©е№ійқўж•°пјҢиҜҘеҖјжҖ»иў«и®ҫзҪ®дёә 1гҖӮ
biBitCount 1CH 2 bytes еғҸзҙ ж·ұеәҰпјҢжҢҮжҳҺдёҖдёӘеғҸзҙ зӮ№йңҖиҰҒеӨҡе°‘дёӘ bit ж•°жҚ® жқҘжҸҸиҝ°пјҢе…¶еҖјеҸҜдёә 1гҖҒ 4гҖҒ 8гҖҒ 16гҖҒ 24гҖҒ 32
biCompression 1EH 4 bytes иҜҙжҳҺеӣҫеғҸж•°жҚ®зҡ„еҺӢзј©зұ»еһӢпјҢеҸ–еҖјиҢғеӣҙеҰӮдёӢпјҡ в‘ 0 вҖ“ RGB ж–№ејҸ в‘Ў1 вҖ“ 8bpp зҡ„ RLE ж–№ејҸпјҢеҸӘз”ЁдәҺ 8bit дҪҚеӣҫ в‘ў2 вҖ“ 4bpp зҡ„ RLE ж–№ејҸпјҢ еҸӘз”ЁдәҺ 4bit дҪҚеӣҫ в‘Ј3 вҖ“ Bit-fields ж–№ејҸ в‘Ө4 вҖ“ д»…з”ЁдәҺжү“еҚ°жңә в‘Ҙ5 вҖ“ д»…з”ЁдәҺжү“еҚ°жңә
biSizeImage 22H 4 bytes иҜҙжҳҺеӣҫеғҸзҡ„еӨ§е°ҸпјҢд»Ҙеӯ—иҠӮдёәеҚ•дҪҚпјҢеҪ“еҺӢзј©зұ»еһӢ дёә BI_RGB ж—¶пјҢеҸҜи®ҫзҪ®дёә 0гҖӮ
biXPelsPerMeter 26H 4 bytes ж°ҙе№іеҲҶиҫЁзҺҮпјҢз”ЁеғҸзҙ /зұіжқҘиЎЁзӨәпјҢжңүз¬ҰеҸ·ж•ҙж•°гҖӮ
biYPelsPerMeter 2AH 4 bytes еһӮзӣҙеҲҶиҫЁзҺҮпјҢз”ЁеғҸзҙ /зұіжқҘиЎЁзӨәпјҢжңүз¬ҰеҸ·ж•ҙж•°гҖӮ
biClrUsed 2EH 4 bytes иҜҙжҳҺдҪҚеӣҫе®һйҷ…дҪҝз”Ёзҡ„еҪ©иүІиЎЁдёӯзҡ„йўңиүІзҙўеј•ж•°гҖӮ
biClrImportant 32H 4 bytes иҜҙжҳҺеҜ№еӣҫеғҸжҳҫзӨәжңүйҮҚиҰҒеҪұе“Қзҡ„йўңиүІзҙўеј•зҡ„ж•° зӣ®пјҢеҰӮжһңжҳҜ 0пјҢеҲҷиЎЁзӨәйғҪйҮҚиҰҒгҖӮ

д»ҺдёҠйқўзҡ„жҸҸиҝ°дҝЎжҒҜпјҢеҶҚжқҘеҜ№з…§ж–Үд»¶ж•°жҚ®пјҡ

0E~11Hпјҡ 0x00000038=56пјҢиҝҷиҜҙжҳҺиҝҷдёӘдҪҚеӣҫдҝЎжҒҜеӨҙзҡ„еӨ§е°Ҹдёә 56 дёӘеӯ—иҠӮгҖӮ

12~15Hпјҡ 0x00000320=800пјҢеӣҫеғҸе®ҪеәҰдёә 800 дёӘеғҸзҙ пјҢдёҺж–Үд»¶еұһжҖ§дёҖиҮҙгҖӮ

16~19Hпјҡ 0x000001E0=480пјҢеӣҫеғҸй«ҳеәҰдёә 480 дёӘеғҸзҙ пјҢдёҺж–Үд»¶еұһжҖ§дёҖиҮҙпјӣиҝҷдёӘж•°жҳҜдёҖдёӘжӯЈж•°пјҢиҜҙжҳҺжҳҜдёҖдёӘеҖ’еҗ‘зҡ„дҪҚеӣҫпјҢд»Җд№ҲжҳҜжӯЈеҗ‘зҡ„дҪҚеӣҫгҖҒд»Җд№ҲжҳҜеҖ’еҗ‘зҡ„дҪҚеӣҫпјҢ иҜҙзҡ„жҳҜеӣҫеғҸж•°жҚ®зҡ„жҺ’еҲ—й—®йўҳпјӣ еҰӮжһңжҳҜжӯЈеҗ‘зҡ„дҪҚеӣҫпјҢ еӣҫеғҸж•°жҚ®жҳҜжҢүз…§еӣҫеғҸзҡ„е·ҰдёҠи§’еҲ°еҸідёӢи§’ж–№ејҸжҺ’еҲ—зҡ„пјҢж°ҙе№іж–№еҗ‘д»Һе·ҰеҲ°еҸіпјҢеһӮзӣҙж–№еҗ‘д»ҺдёҠеҲ°дёӢгҖӮеҖ’еҗ‘зҡ„дҪҚеӣҫпјҢеӣҫеғҸж•°жҚ®еҲҷжҳҜжҢүз…§еӣҫеғҸзҡ„е·ҰдёӢи§’еҲ°еҸідёҠи§’ж–№ејҸжҺ’еҲ—зҡ„пјҢж°ҙе№іж–№еҗ‘дҫқз„¶д»Һе·ҰеҲ°еҸіпјҢеһӮзӣҙж–№еҗ‘ж”№дёәд»ҺдёӢеҲ°дёҠгҖӮ

1A~1BHпјҡ 0x0001=1пјҢиҝҷдёӘеҖјжҖ»дёә 1гҖӮ

1C~1DHпјҡ 0x0010=16пјҢиЎЁзӨәжҜҸдёӘеғҸзҙ еҚ  16 дёӘ bitгҖӮ

1E~21Hпјҡ 0x00000003=3пјҢ bit-fileds ж–№ејҸгҖӮ

22~25Hпјҡ 0x000BB802=768002пјҢеӣҫеғҸзҡ„еӨ§е°ҸпјҢжіЁж„ҸеӣҫеғҸзҡ„еӨ§е°Ҹе№¶дёҚжҳҜ BMP ж–Үд»¶зҡ„еӨ§е°ҸпјҢиҖҢжҳҜеӣҫеғҸж•°жҚ®зҡ„еӨ§е°ҸгҖӮ

26~29Hпјҡ 0x00000EC2=3778пјҢж°ҙе№іеҲҶиҫЁзҺҮдёә 3778 еғҸзҙ /зұігҖӮ

2A~2DHпјҡ 0x00000EC2=3778пјҢеһӮзӣҙеҲҶиҫЁзҺҮдёә 3778 еғҸзҙ /зұігҖӮ

2E~31Hпјҡ 0x00000000=0пјҢжң¬дҪҚеӣҫжңӘдҪҝз”Ёи°ғиүІжқҝгҖӮ

32~35Hпјҡ 0x00000000=0гҖӮ

еҸӘжңүеҺӢзј©ж–№ејҸйҖүйЎ№иў«и®ҫзҪ®дёә bit-filedsпјҲ0x3пјү ж—¶пјҢ дҪҚеӣҫдҝЎжҒҜеӨҙзҡ„еӨ§е°ҸжүҚдјҡзӯүдәҺ 56 еӯ—иҠӮпјҢеҗҰеҲҷпјҢдёә 40 еӯ—иҠӮгҖӮ 56 дёӘеӯ—иҠӮзӣёжҜ”дәҺ 40 дёӘеӯ—иҠӮпјҢеӨҡеҮәдәҶ 16 дёӘеӯ—иҠӮпјҢ йӮЈд№ҲеӨҡеҮәзҡ„ 16 дёӘеӯ—иҠӮж•°жҚ®жҸҸиҝ°дәҶд»Җд№ҲдҝЎжҒҜе‘ўпјҹ зЁҚеҗҺеҶҚеӯҰд№ гҖӮ

1.3В и°ғиүІжқҝ

и°ғиүІжқҝжҳҜеҚ•иүІгҖҒ 16 иүІгҖҒ 256 иүІдҪҚеӣҫеӣҫеғҸж–Үд»¶жүҖжҢҒжңүзҡ„пјҢеҰӮжһңжҳҜ 16 дҪҚгҖҒ 24 дҪҚд»ҘеҸҠ 32 дҪҚдҪҚеӣҫж–Үд»¶пјҢеҲҷ BMPж–Үд»¶з»„жҲҗйғЁеҲҶдёӯдёҚеҢ…еҗ«и°ғиүІжқҝпјҢиҝҷйҮҢе°ұдёҚж·ұе…ҘдәҶи§ЈдәҶгҖӮ

1.4В дҪҚеӣҫж•°жҚ®

дҪҚеӣҫж•°жҚ®е…¶е®һе°ұжҳҜеӣҫеғҸзҡ„ж•°жҚ®пјҢ еҜ№дәҺ 24 дҪҚдҪҚеӣҫпјҢдҪҝз”Ё 3 дёӘеӯ—иҠӮж•°жҚ®жқҘиЎЁзӨәдёҖдёӘеғҸзҙ зӮ№зҡ„йўңиүІпјҢеҜ№дәҺ 16дҪҚдҪҚеӣҫпјҢдҪҝз”Ё 2 дёӘеӯ—иҠӮж•°жҚ®жқҘиЎЁзӨәдёҖдёӘеғҸзҙ зӮ№зҡ„йўңиүІпјҢеҗҢзҗҶпјҢ 32 дҪҚдҪҚеӣҫеҲҷдҪҝз”Ё 4 дёӘеӯ—иҠӮжқҘжҸҸиҝ°гҖӮBMP дҪҚеӣҫеҲҶдёәжӯЈеҗ‘зҡ„дҪҚеӣҫе’ҢеҖ’еҗ‘зҡ„дҪҚеӣҫпјҢдё»иҰҒеҢәеҲ«еңЁдәҺеӣҫеғҸж•°жҚ®еӯҳеӮЁзҡ„жҺ’еҲ—ж–№ејҸпјҢ еҰӮдёӢеҰӮжүҖзӨәпјҲе·Ұиҫ№еҜ№еә”зҡ„жҳҜжӯЈеҗ‘дҪҚеӣҫпјҢеҸіиҫ№еҜ№еә”зҡ„еҲҷжҳҜеҖ’еҗ‘дҪҚеӣҫпјү пјҡ

жүҖд»ҘжӯЈеҗ‘дҪҚеӣҫе…ҲеӯҳеӮЁеӣҫеғҸзҡ„з¬¬дёҖиЎҢж•°жҚ®пјҢд»Һе·ҰеҲ°еҸідҫқж¬Ўеӯҳж”ҫпјҢжҺҘзқҖеӯҳж”ҫз¬¬дәҢиЎҢпјҢдҫқж¬Ўиҝҷж ·пјӣиҖҢеҖ’еҗ‘дҪҚеӣҫпјҢеҲҷе…ҲеӯҳеӮЁеӣҫеғҸзҡ„жңҖеҗҺдёҖиЎҢпјҲеҖ’ж•°з¬¬дёҖиЎҢпјүж•°жҚ®пјҢд№ҹжҳҜд»Һе·ҰеҲ°еҸідҫқж¬Ўеӯҳж”ҫпјҢжҺҘзқҖеҖ’ж•°дәҢиЎҢпјҢдҫқж¬Ўиҝҷж ·гҖӮ

дәҢгҖҒRGB е’Ң Bit-Fields

еҪ“еӣҫеғҸдёӯеј•з”Ёзҡ„иүІеҪ©и¶…иҝҮ 256 з§Қж—¶пјҢе°ұйңҖиҰҒ 16bpp жҲ–жӣҙй«ҳ bpp зҡ„дҪҚеӣҫпјҲ24 дҪҚгҖҒ 32 дҪҚпјүгҖӮи°ғиүІжқҝдёҚйҖӮеҗҲbpp иҫғеӨ§зҡ„дҪҚеӣҫпјҢеӣ жӯӨ 16bpp еҸҠд»ҘдёҠзҡ„дҪҚеӣҫйғҪдёҚдҪҝз”Ёи°ғиүІжқҝпјҢдёҚдҪҝз”Ёи°ғиүІжқҝзҡ„дҪҚеӣҫеӣҫеғҸжңүдёӨз§Қзј–з Ғж јејҸпјҡRGB е’Ң Bit-FieldsпјҲдёӢз§° BFпјү гҖӮ

RGB зј–з Ғж јејҸжҳҜдёҖз§ҚеқҮеҲҶзҡ„жҖқжғіпјҢ дҪҝ RedгҖҒ GreenгҖҒ Blue дёүз§ҚйўңиүІдҝЎжҒҜе®№йҮҸдёҖж ·еӨ§пјҢиӯ¬еҰӮ 24bpp-RGBпјҢе®ғйҖҡеёёеҸӘжңүиҝҷдёҖз§Қзј–з Ғж јејҸпјҢеңЁ 24bits дёӯпјҢдҪҺ 8 дҪҚиЎЁзӨә Blue еҲҶйҮҸпјӣдёӯ 8 дёәиЎЁзӨә Green еҲҶйҮҸпјӣй«ҳ 8 дҪҚиЎЁзӨә RedеҲҶйҮҸгҖӮ

иҖҢеңЁ 32bpp-RGB дёӯпјҢдҪҺ 24 дҪҚзҡ„зј–з Ғж–№ејҸдёҺ 24bpp дҪҚеӣҫзӣёеҗҢпјҢжңҖй«ҳ 8 дҪҚз”ЁжқҘиЎЁзӨәйҖҸжҳҺеәҰ Alpha еҲҶйҮҸгҖӮ32bpp зҡ„дҪҚеӣҫе°әеҜёеӨӘеӨ§пјҢдёҖиҲ¬еҸӘжңүеңЁеӣҫеғҸеӨ„зҗҶзҡ„дёӯй—ҙиҝҮзЁӢдёӯдҪҝз”ЁгҖӮеҜ№дәҺйңҖиҰҒеҚҠйҖҸиҝҮж•Ҳжһңзҡ„еӣҫеғҸпјҢжӣҙеҘҪзҡ„йҖүжӢ©жҳҜ PNG ж јејҸгҖӮ

BF зј–з Ғж јејҸдёҺ RGB дёҚеҗҢпјҢе®ғеҲ©з”ЁдҪҚеҹҹж“ҚдҪңпјҢдәәдёәең°зЎ®е®ҡ RGB дёүеҲҶйҮҸжүҖеҢ…еҗ«зҡ„дҝЎжҒҜе®№йҮҸгҖӮ дҪҚеӣҫдҝЎжҒҜеӨҙд»Ӣз»ҚдёӯжҸҗеҸҠеҲ°пјҢеҪ“еҺӢзј©ж–№ејҸйҖүйЎ№зҪ®дёә BF ж—¶пјҢ дҪҚеӣҫдҝЎжҒҜеӨҙеӨ§е°ҸжҜ”е№іж—¶еӨҡеҮә 16 еӯ—иҠӮпјҢ иҝҷ 16 дёӘеӯ—иҠӮе®һйҷ…дёҠжҳҜ 4дёӘ 32bit зҡ„дҪҚеҹҹжҺ©з ҒпјҢ жҢүз…§е…ҲеҗҺйЎәеәҸпјҢе®ғд»¬еҲҶеҲ«жҳҜ RгҖҒ GгҖҒ BгҖҒ A еӣӣдёӘеҲҶйҮҸзҡ„дҪҚеҹҹжҺ©з ҒпјҢ еҪ“з„¶еҰӮжһңжІЎжңү AlphaеҲҶйҮҸпјҢеҲҷ Alpha жҺ©з ҒжІЎжңүе®һйҷ…ж„Ҹд№үгҖӮ

дҪҚеҹҹжҺ©з Ғзҡ„дҪңз”ЁжҳҜжҢҮеҮә RгҖҒ GгҖҒ B дёүз§ҚйўңиүІдҝЎжҒҜе®№йҮҸзҡ„еӨ§е°ҸпјҢеҲҶеҲ«дҪҝз”ЁеӨҡе°‘дёӘ bit ж•°жҚ®жқҘиЎЁзӨәпјҢд»ҘеҸҠдёүз§ҚйўңиүІеҲҶйҮҸзҡ„дҪҚзҪ®еҒҸз§»йҮҸгҖӮдҫӢеҰӮеҜ№дәҺ 16 дҪҚиүІзҡ„ RGB565 еӣҫеғҸпјҢйҖҡеёёдҪҝз”Ё BF зј–з Ғж јејҸпјҢеҗҢж ·иҝҷд№ҹжҳҜ BF зј–з Ғж јејҸжңҖи‘—еҗҚе’ҢжңҖжҷ®йҒҚзҡ„еә”з”Ёд№ӢдёҖпјҢ е®ғзҡ„ RгҖҒ G е’Ң B еҲҶйҮҸзҡ„дҪҚеҹҹжҺ©з ҒеҲҶеҲ«жҳҜ 0xF800гҖҒ 0x07E0 е’Ң 0x001FпјҢд№ҹе°ұжҳҜ R йҖҡйҒ“дҪҝз”Ё 2 дёӘеӯ—иҠӮдёӯзҡ„й«ҳ 5 дҪҚиЎЁзӨәпјҢ G йҖҡйҒ“дҪҝз”Ё 2 дёӘеӯ—иҠӮдёӯзҡ„дёӯй—ҙ 6 дҪҚиЎЁзӨәгҖӮиҖҢ B йҖҡйҒ“еҲҷдҪҝз”Ё 2дёӘеӯ—иҠӮдёӯзҡ„жңҖдҪҺ 5 дҪҚиЎЁзӨәпјҢ еҰӮдёӢеӣҫжүҖзӨәпјҡ

дёүгҖҒеҰӮдҪ•еҫ—еҲ° 16 дҪҚиүІ RGB565 ж јејҸ BMP еӣҫеғҸпјҹ
